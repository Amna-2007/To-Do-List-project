# 📝 To-Do List App (Python)

Ek simple **console-based To-Do List** program jo pure Python mein likha gaya hai — koi extra library nahi, sirf built-in functions.

---

## 🚀 Kaise Chalayein

```bash
python todo.py
```

---

## 📋 Features

| Option | Kaam |
|--------|------|
| 1 | ✅ Task add karo |
| 2 | 👁️ Tasks dekho |
| 3 | 🗑️ Task delete karo |
| 4 | 👋 Exit |

---

## 🧠 Code Mein Kya Use Hua

| Concept | Kahan Use Hua |
|---------|---------------|
| `list` — `tasks = []` | Tasks store karne ke liye |
| `append()` | Naya task add karne ke liye |
| `pop()` | Task delete karne ke liye |
| `enumerate()` | Tasks ko numbered list mein dikhane ke liye |
| `input()` | User se task ya choice lene ke liye |
| `int()` | Input string ko number mein convert karne ke liye |
| `while True` + `break` | Menu ko baar baar dikhane ke liye |
| Functions (`def`) | `add_task()`, `view_tasks()`, `delete_task()` |
| f-strings | Output mein task ka naam dikhane ke liye |
